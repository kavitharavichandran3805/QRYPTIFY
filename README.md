# QRYPTIFY

Cryptographic Algorithms Identification

This project focuses on automatically identifying cryptographic algorithms used in files, data streams, or communication channels. The system leverages techniques from pattern recognition, statistical analysis, and machine learning to detect and classify commonly used cryptographic algorithms.
